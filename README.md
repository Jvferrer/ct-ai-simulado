# Simulado CT-AI 2.0

Aplicação web estática para estudar para a certificação ISTQB Certified Tester AI Testing (CT-AI) v2.0.

## Como usar

Abra o arquivo `index.html` no navegador e clique em **Iniciar novo simulado**.

## Regras simuladas

- 40 questões de múltipla escolha.
- 44 pontos totais.
- Aprovação com 29 pontos ou mais, equivalente a 65%.
- 60 minutos de duração.
- Opção de 75 minutos para candidato em idioma não nativo.
- Feedback imediato após selecionar a alternativa.
- Alternativas embaralhadas a cada tentativa, mantendo o gabarito correto.
- Dashboard local com execuções, aprovados, reprovados, melhor pontuação e histórico.
- Indicadores de aprendizado com capítulo em que você mais acerta e capítulo para revisar primeiro.

As questões são autorais para estudo e foram alinhadas aos capítulos e objetivos de aprendizagem da syllabus CT-AI v2.0. Elas não são questões oficiais da ISTQB.
